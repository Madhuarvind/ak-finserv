# Arun_Finance

Professional Daily Finance Collection System.

## Project Structure
- `/frontend`: Flutter Mobile & Web application
- `/backend`: Flask REST API with SQLite/PostgreSQL support

## Setup
### Backend
1. `cd backend`
2. `pip install -r requirements.txt`
3. `python app.py`

### Frontend
1. `cd frontend`
2. `flutter pub get`
3. `flutter run`
